Projetos

  • [02/2024 – 06/2024] Projeto de Desenvolvimento de Aplicações Informáticas
    Colaborei como Desenvolvedor no projeto de desenvolvimento de uma aplicação em Java para otimizar rotas de transporte público em Braga. A solução utiliza algoritmos inteligentes e análise de dados de mobilidade para gerar percursos personalizados, visando uma gestão mais eficiente e sustentável, especialmente em zonas de baixa densidade.

  • [09/2024 – 01/2025] Projeto de Engenharia de Dados e Suporte à Tomada de Decisão
    Participei em equipa na implementação de um processo de engenharia de dados utilizando Python 3, Hadoop e Hive para tratar e preparar datasets. Os dados refinados foram utilizados para construir dashboards no Tableau, fornecendo insights para suporte à decisão.

  • [09/2024 – 01/2025] Projeto de Implantação de Aplicações Informáticas Empresariais
    Participei no desenvolvimento em equipa de um projeto de integração entre o SAP S/4HANA e Moloni para a UC de IAIE. O trabalho exigiu a configuração e utilização de serviços OData SAP, consumo de APIs da segunda plataforma, e a implementação de lógica de integração em Python. Utilizei Postman para testar os endpoints e validar os fluxos de dados, seguindo as etapas de análise, modelação, implementação e apresentação da solução.

  • [02/2025 – 06/2025] Projeto de Desenvolvimento Organizacional com Tecnologias de Informação
    No projeto de Desenvolvimento Organizacional com TI (DOTI), apliquei em equipa a metodologia engIS para analisar o estado atual (AS-IS) de uma organização e propor melhorias através de tecnologias de informação. Defini requisitos de TI, avaliei soluções de mercado e planeei a mudança organizacional e tecnológica (TO-BE), resultando na produção de documentação técnica detalhada, desde o diagnóstico aos planos de implementação e análise de benefícios.

  • [02/2025 – 06/2025] Projeto de Programação para a Web
    No projeto de Programação para a Web, desenvolvi em equipa uma plataforma completa para gestão de auditorias e ocorrências na área de Mobilidade e Transportes. Criei as componentes Front-Office (consulta/submissão), Back-Office (gestão operacional) e uma Progressive Web App (PWA para operação no terreno), implementando o fluxo de trabalho associado. A solução, gerida com Git/GitHub, utilizou Vue.js, JavaScript, HTML/CSS, Browser Storage e APIs para fornecer uma ferramenta web funcional e integrada.
